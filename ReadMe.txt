HOW TO USE 

set parameters in main.py :

PROJECT_NAME = 'immo'    -> The folder where data will be store
HOME_PAGE = 'http://www.immoweb.be/fr/'   -> the website you want to crawl
NUMBER_OF_THREADS = 4   -> The numbers of crawlers you want to have. Many crawlers speed up the process, but too much may be difficult too handle for your computer



run : 

"python3 main.py"   to start the crawling
